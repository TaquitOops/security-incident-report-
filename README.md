# Security Incident Report – Network Protocol Analysis

##  Overview
This repository contains a documented cybersecurity incident analysis conducted as part of a security investigation activity. The scenario involves a compromised website that distributed malware to users after a brute force attack on the administrative account.

The purpose of this project is to analyze network traffic, identify the network protocol involved, document the incident professionally, and recommend a security control to prevent similar attacks in the future.

---

##  Incident Summary
- A legitimate website was compromised after an attacker gained administrative access using a brute force attack.
- Malicious JavaScript was injected into the website’s source code.
- Visitors were prompted to download and execute a malicious file.
- Executing the file redirected users to a malicious website and degraded system performance.
- Network traffic analysis confirmed HTTP and DNS activity associated with the redirection.
